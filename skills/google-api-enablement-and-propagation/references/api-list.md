# Google Workspace API map

- Drive folder metadata, listing, and file create: `Google Drive API`
- Google Docs document editing and `batchUpdate`: `Google Docs API`
- Google Sheets create, read, and `values.update`: `Google Sheets API`
- Calendar event listing: `Google Calendar API`
- Gmail API send: `Gmail API`

If an error includes a project number or project ID, enable the API in that same project before retrying.
